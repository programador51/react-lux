# Password (display)

You can pass `showPassword` with a `useState` (or whatever) in order to display the text typed on the input

<Password showPassword />

```tsx
import React from "react";
import { Password } from "react-lux";

function App() {
  return <Password showPassword />;
}

export default App;
```

<hr></hr>

# Password (hide)

You can omit `showPassword` in order to hide the text typed on the input

<Password />

```tsx
import React from "react";
import { Password } from "react-lux";

function App() {
  return <Password />;
}

export default App;
```
